# Project Links

NovaKey is composed of several open-source components. Each repository serves a specific role.

## NovaKey iOS App
- Repository: [https://github.com/osbornepro/NovaKey-iOS-App](https://github.com/osbornepro/NovaKey-iOS-App)
- Description: iOS application for storing and sending secrets securely.

## NovaKey Android App
- Repository: [https://github.com/osbornepro/NovaKey-Android-App](https://github.com/osbornepro/NovaKey-Android-App)
- Description: Android client with equivalent functionality to the iOS app.

## NovaKey-Daemon
- Repository: [https://github.com/osbornepro/NovaKey-Daemon](https://github.com/osbornepro/NovaKey-Daemon)
- Description: Desktop service that receives secrets from paired devices and injects them securely.

## NovaKeyKEMBridge
- Repository: [https://github.com/osbornepro/NovaKeyKEMBridge](https://github.com/osbornepro/NovaKeyKEMBridge)
- Description: Cryptographic bridge library used for key encapsulation and secure channel setup.
