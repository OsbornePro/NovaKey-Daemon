# Project Links

NovaKey is composed of several open-source components. Each repository serves a specific role.

## NovaKey iOS App - iPhone Application
- Repository: https://github.com/your-org/NovaKey-iOS-App
- Description: iOS application for storing and sending secrets securely.

## NovaKey Android App - Android Application (*In Progress Still*)
- Repository: https://github.com/your-org/NovaKey-Android-App
- Description: Android client with equivalent functionality to the iOS app.

## NovaKey-Daemon - Service that gets installed on computers to create a listener
- Repository: https://github.com/your-org/NovaKey-Daemon
- Description: Desktop service that receives secrets from paired devices and injects them securely.

## NovaKeyKEMBridge - Client that talks the service/daemon on a computer
- Repository: https://github.com/your-org/NovaKeyKEMBridge
- Description: Cryptographic bridge library used for key encapsulation and secure channel setup.
